### Built-in Functions (Types and Conversions) {#pre-parse}

`isNaN(num)`:
  ~ Safely test if `num` is `NaN`

`isFinite(num)`:
  ~ Test if `num` is **not** `NaN` or `Infinity`

`parseInt(str)`:
  ~ Convert a string to a number (integer)

`parseFloat(str)`:
  ~ Convert a string to a number (float)
