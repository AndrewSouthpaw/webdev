## Web Workers

### Web Worker Basics

  * Allows you to start a new background "thread"

  * Messages can be sent to and from the worker

  * Message handling is done through events

  * Load scripts with: `importScripts("name.js");`

### Browser Support

  - IE      >= 10
  - Firefox >= 3.5
  - Safari  >= 4
  - Chrome  >= 4
  - Opera   >= 10.6
