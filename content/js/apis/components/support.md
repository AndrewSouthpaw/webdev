### Browser Support ###

  * Custom Elements and Templates

    - IE (No support)
    - Edge (No support)
    - Firefox >= 63 (2018)
    - Safari  >= 10.1 (2017)
    - Chrome  >= 53 (2016)

  * Shadow DOM

    - IE (No support)
    - Edge (No support)
    - Firefox >= 63 (2018)
    - Safari >= 11.1 (2018)
    - Chrome  >= 66 (2018)

(Polyfills [exist for most browsers][polyfills].)

[polyfills]: https://www.webcomponents.org/polyfills
